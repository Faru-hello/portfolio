Personal Portfolio Website

A responsive personal portfolio website built using HTML, CSS, and JavaScript to showcase my background, skills, projects, and research publication. 
This portfolio is designed to give recruiters and collaborators a clear overview of my profile and ongoing work.
Live Website: https://faru-hello.github.io/portfolio/
About the Project

This portfolio website serves as a professional online presence. It highlights:

1.My academic and technical background

2.Real projects and ongoing work

3.An IEEE conference publication

4.Contact information for collaboration or opportunities

The website is lightweight, fast, and fully responsive, making it accessible across desktop and mobile devices.
Tech Stack

Deployment: Netlify / GitHub Pages

Version Control: Git & GitHub
